Spotify Ad Muter The program when launched will open Spotify. Login and go to the spotify web player

When an Ad is played your computer will me muted.

After the ad is gone it will unmute your computer. The launch file is MainSpotify.py

Tested on Google Chrome

Python Libraries Needed PyCaw and Selenium

You will need the Chrome Driver installed to run this program.

Make sure that the chromedriver you have matched to your Chrome version, you can check this by going to chrome settings and about Chrome.

Make sure that the chrome driver path is "C:\Program Files (x86)\chromedriver.exe"
